# TimeMind AI - Frontend

هذا هو الجزء الأمامي (Frontend) من نظام TimeMind AI.

## التثبيت السريع

```bash
# تثبيت الحزم
npm install

# نسخ ملف البيئة
cp .env.example .env

# تشغيل السيرفر
npm run dev
```

## البناء للإنتاج

```bash
npm run build
```
